## How to use


#### cd to the backend folder and  put this into terminal
```npm
 pip install -r requirements.txt

```
#### to run backend 
```npm
 python manage.py runserver

```
#### cd to the frontend folder and  put this into terminal
```npm
 npm install

```
#### to run the frontend app
```npm
npm start

```
